# FASTBOOKS · 링크 중앙 관리 (links.md)

이 파일의 `= 오른쪽 주소`만 바꾸면, 해당 섹션의 버튼 링크가 자동으로 바뀝니다.
- 왼쪽 키(예: `hero.shop`)는 **절대 바꾸지 마세요.**
- `- 키 = 주소` 형식과 띄어쓰기를 그대로 두세요.
- 주소 뒤 `#` 부터는 설명(주석)이라 무시됩니다.
- 적용/설치 방법은 `LINK-SETUP.md` 를 보세요.

## 히어로 (hero.html)
- hero.shop  = #books   # 상품 둘러보기 / 책자 주문 (슬라이드 3개 공통)
- hero.quote = #quote   # 별도견적문의 (슬라이드 3개 공통)

## 무선제본 활용 (usecases-perfect.html)
- ucp.shop  = #books    # 무선책자로 만들기 (3개 공통)
- ucp.quote = #quote    # 별도견적문의 (3개 공통)

## 중철제본 활용 (usecases-saddle.html)
- ucs.shop  = #books    # 중철책자로 만들기 (3개 공통)
- ucs.quote = #quote    # 별도견적문의 (3개 공통)

## 상품 카탈로그 (products.html) — 각 상품 "주문하기"
- prod.spring   = #quote   # 스프링책자
- prod.perfect  = #quote   # 무선책자
- prod.saddle   = #quote   # 중철책자
- prod.leaflet  = #quote   # 리플렛
- prod.flyer    = #quote   # 전단지
- prod.postcard = #quote   # 엽서

## 별도견적 (quote.html) — Fluent Forms 견적 요청서로 이동
- quote.form = #   # 견적 요청서(Fluent Form) 주소. 왼쪽 버튼 + 오른쪽 카드 전체가 이 주소로 이동

## CTA — 신규회원 가입 유도 (cta.html)
- cta.signup = #   # 회원가입 페이지 (예: /my-account/ 또는 회원가입 폼 주소)
- cta.login  = #   # 로그인 페이지 (예: /my-account/)
